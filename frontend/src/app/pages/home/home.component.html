Orbit Project

Este proyecto presenta el diseño de un sistema que simula un satélite y su comunicación con una estación de tierra, combinando adquisición de datos, control y visualización en tiempo real. El sistema permite monitorizar variables ambientales, seguir la posición del satélite, enviar comandos de forma remota y garantizar la fiabilidad de la información transmitida. La solución se basa principalmente en Arduino Uno como plataforma de control y utiliza Arduino (C/C++) y Python para el desarrollo del software, integrando hardware y software en un proyecto funcional y educativo.

Monitorización Ambiental en Tiempo Real

El satélite incorpora sensores que permiten medir temperatura, humedad y presión atmosférica de forma continua. Los datos se transmiten a la estación de tierra, donde se procesan y se representan mediante gráficas claras que facilitan el análisis y la comprensión de las condiciones ambientales registradas durante la misión.

Detección de Distancias y Movimiento Controlado

El sistema integra un sensor de ultrasonidos junto a un mecanismo de movimiento controlado, capaz de detectar objetos y medir distancias con precisión. Esta información se visualiza en la estación de tierra mediante representaciones gráficas intuitivas, ofreciendo una interpretación rápida del entorno detectado.
